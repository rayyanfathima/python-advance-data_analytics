# Python Practice Programs

This repository contains a collection of beginner-friendly Python
programs demonstrating fundamental programming concepts such as
functions, file handling, dictionaries, conditional logic, loops, and
user input processing. These scripts are ideal for students, educators,
and anyone beginning with Python.

## Contents Overview

### 1. Hello Name Program

A simple function-based script that prints a greeting message for a
given name. Source: helloname.py

### 2. File Handling -- Version 1

Reads a CSV file (sheet1.csv), splits lines into lists, converts numeric
values, and stores the processed data in a dictionary. Source:
filehandle.py

### 3. File Handling -- Version 2

A simplified version of the file-handling program that extracts fixed
columns from the CSV file and stores them into a dictionary. Source:
filehandle2.py

### 4. ATM Simulation Program

A console-based ATM application featuring: - PIN verification - Balance
enquiry - Withdrawal process - Loop-based menu system Source: atm.py

### 5. Employee Salary Categorization

Processes an employee salary dictionary and classifies salaries into 1
(above threshold) or 0 (below threshold). Source: salary15.py
